During the encode and decode process we perform the following on x : ( ( (x + 13) % 26 ) + 13 ) % 26 Since modulus is a distributive function this can be re-written as : x % 26 + (13 + 13) % 26 Since 13 + 13 is 26 the end of this function is 0 and we end up with our original location x.
